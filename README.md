## Description
A command line tool for displaying "Today is what day" from [一般社団法人 日本記念日協会](https://www.kinenbi.gr.jp/)

## Usage
```
$ whatday
```

## Install

To install, use `go install`:

```bash
$ go install github.com/valbeat/whatday@latest
```
